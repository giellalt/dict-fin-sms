# Documentation for the Finnish  ⇒ Skolt Sámi dictionary

The source code of the Finnish ⇒ Skolt Sámi dictionary is **not**
edited in this folder (../src) but in a dictionary page run by Jack
Rueter and thereafter exported here. 

The Skolt Sámi ⇒ Finnish dictionary, on the other hand, has source
files in `../../lang-sms/src/fst/morphology/stems`.
